\\Course_Information

Welcome to CSCB20: Introduction to Databases and Web Applications. This course gives an introduction to:

Databases:

  • Terminology and applications;
  • Creating, querying, and updating databases

Web documents and applications:

  • Static and interactive documents
  • Web servers and dynamic server-generated content
  • Web application development and interface with databases

\\Prerequisites

Experience programming in an imperative language such as Python, Java, or C. This course may not be taken after - or concurrently with - any C- or D-level CSC course.

Exclusion: CSC343H
